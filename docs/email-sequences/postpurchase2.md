# Post-Purchase Email 2 - Cross-Sell

**Sequence:** Post-Purchase
**Number:** 2 of 3
**Send delay:** Day 7
**Subject line:** Something your family might love next
**Preview text:** Based on what you got - one suggestion

---

> **How this works:** The website automatically tags each buyer with a `cross-sell:product-slug` tag based on what they purchased. In Kit, create **one email per cross-sell tag** inside the Post-Purchase automation, using Kit's conditional/filter step. Each version below is ready to paste. The buyer only receives the one that matches their tag.

## Cross-sell tags the website sends:

| Nature pack | `cross-sell:seasonal-bundle` | Seasonal Bundle ($49.99) |
